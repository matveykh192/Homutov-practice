Введение

Наш проект представляет собой инновационную платформу для преобразования жестовой коммуникации в цифровой текст и речь. Разрабатываемая система использует современные технологии компьютерного зрения и машинного обучения для распознавания жестов языка глухих в реальном времени. Мы создаем не просто переводчик, а полноценную коммуникационную экосистему, веб-интерфейс и API для интеграции с другими сервисами. Проект направлен на разрушение барьеров в общении между слабослышащими и слышащими людьми. Платформа будет полезна в образовательных учреждениях, социальных службах и бизнес-среде. Особое внимание уделяется простоте интерфейса, чтобы сделать систему доступной для пользователей всех возрастов.

Основные цели

1. Создание первого в мире массового сервиса жестового перевода в реальном времени
Разработать платформу, которая без специализированного оборудования (только камера смартфона/ноутбука) переводит жестовый язык в текст и голос с задержкой менее 0.5 секунды. Точность распознавания — не менее 95% для базового набора жестов и 85% для сложных комбинаций с учетом индивидуальных особенностей пользователя.

2. Преодоление социальной изоляции глухих и слабослышащих

3. Создание качемтвенной 3D модели которая будет показывать жесты.
